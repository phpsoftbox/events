# DI

Пример конфигурации с PHP-DI:

```php
use PhpSoftBox\Events\EventDispatcher;
use Psr\Container\ContainerInterface;

return [
    EventDispatcher::class => static function (ContainerInterface $container): EventDispatcher {
        $dispatcher = new EventDispatcher(container: $container);

        return $dispatcher;
    },
];
```

Очередь подключается через `setQueue()` или при создании `EventDispatcher`.
